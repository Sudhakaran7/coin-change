You are given coins of different denominations and a total amount of money. 
Write a function to compute the number of combinations that make up that amount. You may assume that you have infinite number of each kind of coin.

Input Description:
First line contains, n and amount as integers. (1<n,amount<10000)
Second line contains, coins a integer array with size n.

Sample Input:
3 5
1 2 5

Sample Output:
4

Explanation:
5=5
5=2+2+1
5=2+1+1+1
5=1+1+1+1+1
